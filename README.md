# My Metakit Project

Content managed by [Metakit](https://metakit.app) — a visual MDX editor that commits straight to your repo.

## Structure

```
content/
├── blog/        # Blog posts
├── docs/        # Documentation
└── changelog/   # Release notes
```

## Getting started

1. Open this repo in [Metakit](https://metakit.app)
2. Edit content visually or in code
3. Every save commits to this repo
